Згенеруй API-документацію у форматі Postman Collection (v2.1.0) 
на основі коду контролера нижче.

Вимоги:
- опис колекції: модель даних, таблиця статус-кодів, seed-дані, базовий URL;
- усі запити в одній папці;
- для кожного ендпоінта: markdown-опис, тіло запиту, path-змінні (:id), 
  приклади відповідей (успішна + 404);
- реалістичні україномовні дані;
- виведи лише валідний JSON у код-блоці.
